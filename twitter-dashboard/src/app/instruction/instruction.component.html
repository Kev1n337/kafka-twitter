Welcome to Twitter Analysis

This App uses the following technologies, to analyze the latest Tweets of certain topics:
  • Apache Kafka
  • KSQL
  • KSQL REST Client
  • Kafka Connect Twitter
  • Express for serverside routing
  • Angular 6 Client
  • HTTP and WebSocket communication between server and client
Using three different Kafka Topics/Streams, the top tweeting users and most used hashtags are displayed. Additionally each Topic contains a feed with the newest tweets. Please follow the steps specified in the repository's README to get the app running on your machine.
